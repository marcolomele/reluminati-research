## Purpose

Miscellaneous scripts for operations related to data.

## Scripts

### Data Processing

- **`download_and_process_data.py`** - Downloads EgoExo4D videos, extracts annotated frames, and organizes them for training. Supports cooking and health scenarios.

- **`process_data.py`** - Loads EgoExo4D videos, extracts annotated frames, and organizes them for training. Supports all scenarios.

- **`sample_takes`** – Sample a fraction of takes from an existing split.json stored at --root and returns .json with sampled UIDs per each split.
